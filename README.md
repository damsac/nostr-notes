# nostr-notes

Nostr note reader built with [RMP](https://github.com/damsac/rmp) — Rust core + UniFFI + SwiftUI + Compose.

Connects to `wss://nostr.damsac.studio` relay.

## Build

```
just test          # Run Rust tests
just ios-build     # Build iOS (requires macOS)
just android-build # Build Android
```
